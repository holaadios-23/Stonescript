· ·:·: Stonescript (Beta) :·:· ·
v4.28.0 - 2026/01/20

Stonescript es el lenguaje utilizado en la Piedra Mental para automatizar el equipo. 
Es un lenguaje minimalista pero potente dentro de Stone Story RPG.

===============================================================================
1. BÁSICOS
===============================================================================
?      : Evalúa una condición lógica (if).
:      : Rama lógica alternativa (else).
:?     : Rama lógica alternativa con condiciones (else-if).
//     : Comentario de una línea.
/* */  : Bloque de comentario.
^      : Continúa la línea anterior.

===============================================================================
2. ESTADO DEL JUEGO (GAME STATE)
===============================================================================
loc.id / loc.name    : ID o nombre de la ubicación actual.
loc.stars            : Dificultad actual (estrellas).
loc.begin            : Verdadero solo en el primer frame de una ubicación.
loc.loop             : Verdadero en el primer frame después de un bucle de Ouroboros.
screen.w / screen.h  : Ancho y alto de la pantalla de juego.
player.name          : Nombre del personaje.
foe                  : Enemigo objetivo actual.
foe.id / foe.name    : Identificador o nombre del enemigo.
foe.hp / foe.maxhp   : Puntos de vida actuales/máximos del enemigo.
foe.armor            : Armadura del enemigo.
foe.distance         : Distancia entre el jugador y el enemigo.
foe.count            : Número de enemigos en un rango de 46 unidades.
hp / maxhp           : Puntos de vida actuales/máximos del jugador.
armor / maxarmor     : Armadura actual/máxima del jugador.
item.left / item.right : Ítem equipado en la mano izquierda o derecha.
buffs.string         : Información de todos los efectos positivos en el jugador.
debuffs.string       : Información de todos los efectos negativos en el jugador.
pos.x, pos.y, pos.z  : Posición actual del jugador.
time                 : Frame actual de la ubicación.
time.second          : Segundo actual de la ubicación (0-59).
time.msbn            : Timestamp Unix en milisegundos (usar con BigNumber).
totaltime            : Frame acumulado de la ubicación (incluyendo sub-ubicaciones).
item.GetCount(id)    : Cantidad de un ítem en inventario (ej: item.GetCount("candy")).
res.stone/wood/tar...: Recursos actuales en el inventario.
rng                  : Número aleatorio entre 0 y 9999.

===============================================================================
3. COMANDOS
===============================================================================
activate (ability)   : Activa una habilidad (potion, P, left, L, right, R).
brew (ingredientes)  : Recarga la poción (ej: brew bronze + tar). Solo en time = 0.
equip (str)          : Equipa un ítem (ej: equip vigor crossbow *8 +5).
equipL / equipR      : Equipa ítem en la mano izquierda o derecha específicamente.
loadout (n)          : Equipa el set de equipo número 'n'.
> (str)              : Imprime texto en la parte superior de la pantalla.
>oX,Y,#color,text    : Imprime texto relativo a la posición del jugador.
>`X,Y,#color,text    : Imprime texto relativo a la esquina superior izquierda.
var (nombre)         : Declara una variable.
func (nombre)        : Declara una función.
import (script)      : Carga y ejecuta un script externo (.txt).
play (sonido)        : Reproduce un efecto de sonido.
disable/enable loadout print : Oculta o muestra el texto de cambio de equipo.
disable/enable hud   : Oculta o muestra elementos de la interfaz.

===============================================================================
4. FILTROS DE BÚSQUEDA Y COMPARACIONES
===============================================================================
Elementos: poison, vigor, aether, fire, air, ice.
Tipos: arachnid, serpent, insect, machine, boss, flying, spawner.
Niveles: *[número] (estrellas), +[número] (encantamiento).

Comparadores:
=  : Igual a / El string contiene.
!  : No igual / El string no contiene.
&  : Operador lógico Y (AND).
|  : Operador lógico O (OR).
> / < / >= / <= : Comparaciones numéricas.

===============================================================================
5. OPERACIONES MATEMÁTICAS
===============================================================================
+ (suma), - (resta), * (multiplicación), / (división - redondea abajo).
++ (incremento), -- (decremento), % (módulo - resto).
() : Paréntesis para prioridad.
!  : Negación (invierte booleano).

===============================================================================
6. FUNCIONES NATIVAS (APIs)
===============================================================================
--- MATH ---
math.Abs(n), math.Floor(n), math.Ceil(n), math.Clamp(n, min, max).
math.Sin(rad), math.Cos(rad), math.Sqrt(n), math.Pi.
math.BigNumber(str/n): Para manejar números extremadamente grandes.

--- STRING ---
string.Size(s), string.Sub(s, start, len), string.Equals(s1, s2).
string.Split(s, sep), string.Join(sep, array), string.ToUpper(s).

--- STORAGE ---
storage.Set(key, val) : Guarda un valor permanentemente entre runs.
storage.Get(key)      : Recupera un valor guardado.

--- BUFFS ---
buffs.GetTime(id)     : Devuelve los frames restantes de un buff específico.

--- DRAW ---
draw.Clear()          : Limpia la pantalla.
draw.Bg(x, y, color)  : Cambia el color de fondo de un glifo.
draw.Box(x, y, w, h, col, style) : Dibuja un rectángulo.

--- UI (INTERFAZ DE USUARIO) ---
ui.Clear()            : Elimina todos los elementos y reinicia la UI.
ui.OpenMind()         : Abre la interfaz de programación de la Piedra Mental.

PROPIEDADES COMUNES (Todos los componentes):
 .x, .y      : Posición.
 .w, .h      : Dimensiones.
 .visible    : Mostrar/Ocultar (true/false).
 .color      : Color hex (ej: #FF0000).
 .anchor     : Punto de origen (top_left, center, bottom, etc).
 .dock       : Ajuste al contenedor padre.

COMPONENTES ESPECÍFICOS:

1. ui.AddPanel()      : Contenedor visual.
    .style            : 0 (invisible), 1 (borde simple).

2. ui.AddText(str)    : Etiqueta de texto.
    .text             : Contenido de texto.
    .align            : center (0), left, right.

3. ui.AddButton()     : Botón interactivo.
    .text             : Texto del botón.
    .hover            : Verdadero si el mouse está encima.
    .SetPressed(func) : Define la función a ejecutar al hacer clic.

4. ui.AddAnim(ascii)  : Animación basada en texto.
    .duration         : Frames que dura un ciclo.
    .loop             : Si debe repetirse (true/false).
    .Play() / .Stop() : Control de reproducción.

5. ui.AddCanvas()     : Lienzo para dibujo avanzado (pixel a pixel).
    .Set(x, y, col, char) : Dibuja un carácter con color en una coordenada.

===============================================================================
7. ARRAYS Y BUCLES
===============================================================================
var a = [1, 2, 3]     : Inicializa un array.
a.Add(val)            : Añade al final.
a.Emplace(index, val) : Reemplaza o inserta en un índice específico.
a.Count()             : Tamaño del array.
a.RemoveAt(index)     : Elimina elemento.
for i = a..b          : Bucle de un rango numérico.
for val : array       : Bucle a través de los elementos de un array.

Acceso: var x = a[0] (primer elemento).

8. ENTRADA PERSONALIZADA (CUSTOM INPUT)

===============================================================================
CONSEJOS RÁPIDOS
===============================================================================
- La indentación (espacios) importa después de un '?'.
- El script se ejecuta 30 veces por segundo (30 FPS).
- Mantener 'Tab' presionado en el juego muestra errores y estado actual.
- Los comandos de equipar ('equip') ejecutados al final de un frame tienen prioridad.
